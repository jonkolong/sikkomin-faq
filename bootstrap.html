FAQ?

Bagaimana cara mendapatkan akun?

Untuk mendapatkan akun, Anda bisa mendaftar menggunakan nomor WhatsApp melalui admin Sikkomin DS pada unit kerja Anda.

Bagaimana cara mendapatkan token login?

Token dikirimkan melalui WhatsApp setiap kali Anda login menggunakan nomor WhatsApp yang sudah terdaftar. Catatan:
 Pastikan perangkat Anda terhubung ke internet

Bagaimana cara mengganti nomor WhatsApp terdaftar di akun saya?

Anda bisa mengganti nomor WhatsApp terdaftar di akun Anda melalui admin pada unit kerja Anda. Catatan:
 Data yang sudah anda buat tetap tersimpan di database aplikasi

Bagaimana cara reset "passphrase" sertifikat tanda tangan elektronik?

Silahkan hubungi Bidang TIKSan seksi Persandian pada Dinas Kominfostan. Catatan:
 Tidak boleh diwakilkan

Bagaimana cara validasi TTE laporan bulanan saya?

Validasi TTE laporan bulanan dapat diakses melalui opsi Periksa TTE pada menu Berkasku. Catatan:
 Jika valid, maka file berisi nama 2 (dua) orang
